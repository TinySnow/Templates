### add-head.sh

#### 用途

- 为每个 `.md` 文件在文件首行添加文件名。
- 第二个参数为路径。
- 处理第二个参数下所有的 `.md` 文件。

#### 示例

- 文件名为：Test.md.
- 文件内容为：Test content.

|    修改前     |          修改后           |
| :-----------: | :-----------------------: |
| Test content. | # Test<br />Test content. |
